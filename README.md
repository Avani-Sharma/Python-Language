## 🧠 Overview
This repository contains my **Python practice programs** — organized topic-wise to help understand the fundamentals of programming.  
It includes everything from basic syntax and input/output to OOPs, data structures, and pattern printing.  
Perfect for beginners who want to revise or learn Python step-by-step. 🚀
